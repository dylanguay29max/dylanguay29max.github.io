The Parts You Will Need To Build A PC

Homepage

In order to build a PC, you will need these specific parts:

  1. GPU (Graphics card)
  2. CPU (Computer Processing Unit)
  3. CPU cooler (Cools down CPU)
  4. Motherboard (Connects many of the parts)
  5. Ram (Memory)
  6. Storage (Hard drive or SSD)
  7. PSU (Power supply unit)
  8. Case (Put all the components inside the case)
